# Скриншоты для диссертации

Папка содержит печатные PNG-скриншоты без браузерной рамки и без лишних декоративных подписей. Основные новые материалы: приложение, Lighthouse-отчеты, summary/comparison/budget, min/median/max-статистика и фрагменты конфигурации.

Использовать в отчете прежде всего файлы screen_01-screen_05, screen_16-screen_20, screen_30-screen_31.
